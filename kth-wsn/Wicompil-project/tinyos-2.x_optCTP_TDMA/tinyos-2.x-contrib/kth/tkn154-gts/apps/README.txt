Description
--------------------------------------------------

This folder contains test applications for "TKN15.4" with the GTS
implementation. To test the GTS is only possible the use of beacon-enabled
mode which is located in the "beacon-enabled" folder. Every test
application resides in a separate subdirectory which includes a README.txt
describing what it does and how it can be installed. More information can be
found in doc/pdf/
